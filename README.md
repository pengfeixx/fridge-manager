# Fridge Manager

冰箱食材管家 App —— 基于 Flutter 的家庭冰箱食材管理工具。

## 功能概览

- 拍照识图录入食材（AI 主入口），支持手动/语音录入
- 按保质期管理食材，临期提醒
- 按快过期食材推荐菜谱
- 家庭成员与忌口/过敏管理
- 基于《中国居民膳食指南》的营养分析与买菜建议
